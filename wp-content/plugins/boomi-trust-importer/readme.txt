== Description ==

Import statuses from the old trust site.

== Installation ==

1. Upload the `boomi-trust-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, click on Boomi Trust Statuses
